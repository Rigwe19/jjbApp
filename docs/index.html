Getting Started

Open your favourite command prompt and run $git clone https://github.com/Rigwe19/jjbApp.git to clone this repository

Then run
$npm install
$ionic serve --external

API Address

The api domain is already stored in state so all you will need to do is to get it with recoil

const href = useRecoilValue(urlAtom);

Using the Capacitor HTTP request plugin

Request are made using the capacitor HTTP plugin which can be imported

import { Http } from '@capacitor-community/http'; 
                            
Http.request({
    method: "POST",
    url: href + "/api/login",
    headers: {
        'Content-Type': 'application/json',
        'Data-Type': 'json',
        'accept': 'application/json'
    },
    data: data
    }).then((response) => {
        if (response.data.success) {
            // do something
        } else if (response.status === 422) {
            // error 422 are validation errors
        }else{
            // return response.data.error
        }
    }).finally(() => {
        // run another code
    });
