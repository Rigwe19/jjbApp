Login

to use this api we need to get an auth token from the api and this can be achieved by login into the system

from the app we use the login route from the ionic HTTP library

url: href + "/api/login",

Response

On successful login it returns an object response.data.user 

                            
{
    employee_id: number,
    department_id: number,
    name: string,
    passport: string,
    position: string,
    shift: string,
    department: string,
    created_at: string,
    brand: string,
    id: number,
    role: number,
    token: string,
    updated_at: string,
    username: string,
    location_id: number,
    setOpening: boolean,
    setClosing: boolean,
    isLoggedIn: boolean,
}

this value would be save in state for future use
