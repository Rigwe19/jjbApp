This page document the accounting endpoints

Dashboard

Get Dashboard | GET

Dashboard data can been retrieved with the following endpoint

/api/account/get/dashboard

it returns the following

Variable Description Properties
success returns a boolean depending on if the API call is successful or not boolean
dashboard return an object containing dashboard data { available: number; payment: number; sales: number; expenses: number; purchase: number; total: number; }
Search Dashboard | POST

Dashboard data can been search with the following endpoint, it returns the same data as get dashboard

/api/search/account/dashboard

the post request accept the following request data

Variable Description Properties
date the date for the query to search 2023-07-29T09:55:15.110Z
type the type of interval {1: daily, 2: weekly, 3: monthly, 4: yearly, 5: custom}
start | optional when the type is custom the start date should be supplied 2023-07-29T09:55:15.110Z
end | optional when the type is custom the end date should be supplied 2023-07-29T09:55:15.110Z

Sales

Get Sales | GET

Sales can been retrieved with the following endpoint

/api/account/get/sales

it returns the following

Variable Description Properties
success returns a boolean depending on if the API call is successful or not boolean
expenses return an array of object containing all the data related to expenses { id: number; name: string; type: string; purchase: 0|1; }[]
payment_mode return an array of object of mode of payment { id: number; mode: string; balance: number; }[];
brands return list of available brands { id: number; name: string; }[];
units return list of SI units for unit conversions { id: number; name: string; unit: string; // SI unitbase: string; // lowest form for the SI unit}[];
suppliers returns a list of suppliers { id: number; name: string; code: string; phone: string; address: string; location_id: number; }[];
locations returns a list of locations { id: number; name: string; state_id: number; }[];
accounts returns a list of accounts { id: number; name: string; logo: string; }[];
purchases returns a list of purchases { id: number; date: string; location_id: number; purchase_id: number; expense_id: number; supplier_id: number; quantity: number; unit_weight: number; unit_id: number; unit_price: number; amount: number; paid: number; total_weight: number; brand_id: number; mode_id: number; reason: string|null; bank_id: number; delivered: 0|1; name: string; }[];
banks returns a list of banks { id: number; bank_name_id: number; acc_number: string; acc_name: string; bank: string; acc_balance: number; logo: string; }[];
